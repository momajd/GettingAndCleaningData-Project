# Codebook


| Abbreviation   | Variable                              |
| :------------- | :------------------------------------ |
| t              | time domain signal                    |
| f              | fast fourier transform (FFT) applied  |
| Bd             | Body                                  |
| Grav           | Gravity                               |
| Acc            | Acceleration                          |
| Jrk            | Jerk                                  |
| Mag            | Mag                                   |
| Mean           | mean                                  |
| Std            | standard deviation                    |
| X              | X direction                           |
| Y              | Y direction                           |
| Z              | Z direction                           |

ex: tGravAccXMean --> time domain signal, Gravity, Acceleration, X direction, Mean
